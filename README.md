# RL.SYS HFT - Terminal Quantitativo Institucional

**RL.SYS HFT** não é um bot de apostas. É um Terminal de High-Frequency Trading (HFT) e Engenharia Quantitativa projetado especificamente para mapear, analisar e explorar ineficiências matemáticas na Roleta Europeia ao vivo.

O sistema atua como um escudo psicológico e financeiro, isolando o operador do *tilt* emocional através de gestão de margem rigorosa, machine learning Bayesiano e análise estatística avançada.

## 🎯 Arquitetura de Elite (Core Features)

* **Gestão de Capital (Bankroll Management):** Implementação dinâmica do Critério de Kelly Fracionário. O sistema calcula o tamanho exato da ficha baseado no seu P&L atual e no Risco/Retorno (Payout) da matriz, protegendo contra ruína matemática.
* **Cofre Global Diário:** Sistema de Hard-Stop. Atingiu -15% de perda ou +25% de lucro? O sistema trava a mesa. Operações hostis são sumariamente bloqueadas.
* **Visão Física (Heatmap Balístico):** Varredura de cilindro físico usando "Sliding Windows". Identifica clusters (anomalias de agrupamento de bolas) com base no Z-Score.
* **Inteligência de Mercado (Índice VIX & Lei do Terço):** Cálculo de Entropia de Shannon em tempo real para medir o "Caos" da mesa. Combinado com o monitoramento da Lei do Terço (Dispersão vs. Repetição) para vetar entradas perigosas.
* **Máquina do Tempo (Backtest Lab):** Motor de testes No-Code. Crie matrizes operacionais cruzando dados temporais ou físicos e teste-as contra o Big Data do seu próprio histórico em milissegundos.
* **Reconhecimento de Terreno (OCR Sandbox):** Validador isolado de mesa que extrai dados via Visão Computacional, submetendo o histórico a um Filtro Darwiniano para cravar `GO` (Mesa Segura) ou `NO GO` (Mesa Hostil).

## 🛠️ Stack Tecnológico
* **Frontend:** React, TypeScript, TailwindCSS, Framer Motion, Recharts.
* **Backend:** Node.js, Prisma ORM, SQLite.
* **Cognitivo:** Pesos Bayesianos, Cadeias de Markov, Entropia de Shannon, OCR Integrado.

## 🚀 Instalação e Inicialização

1. Clone o repositório
git clone https://github.com/seu-usuario/rlsys-hft.git

2. Instale as dependências
npm install

3. Inicie o Banco de Dados
npx prisma db push
npx prisma generate

4. Inicie o Servidor e o Terminal
npm run build
npm run start
